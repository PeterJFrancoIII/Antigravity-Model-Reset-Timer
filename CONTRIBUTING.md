# Contributing to Antigravity Account Timer

First off, thank you for considering contributing! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tab to report bugs.
- Include a clear title and description, as much relevant information as possible, and a code sample or an executable test case demonstrating the expected behavior that is not occurring.

### Suggesting Enhancements
- Enhancement suggestions are tracked as GitHub issues.
- Provide a step-by-step description of the suggested enhancement.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the test suite passes.
4.  Make sure your code lints.
5.  Issue that pull request!

## Development Setup
1.  Clone your fork.
2.  Install requirements: `pip install -r requirements.txt`.
3.  Run the launcher: `python3 launcher.py`.

## Style Guide
- Follow PEP 8 for Python code.
- Ensure CSS follows the project's glassmorphism aesthetic.
- Keep the UI "Ultra Smart but Ultra Simple".
